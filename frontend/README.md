# SparkLabs 前端编辑器

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 技术栈

- **React 18** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库

## 功能模块

1. **叙事编辑器** - 故事文本编辑和 AI 分析
2. **资产生成器** - AI 驱动的角色和场景生成
3. **语音合成器** - 多角色音频生成
4. **分镜编辑器** - 故事板和分镜头设计
5. **视频合成器** - 最终视频渲染和导出
